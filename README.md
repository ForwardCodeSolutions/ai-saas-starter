# ai-saas-starter

Full-stack AI SaaS boilerplate with FastAPI backend, React/Next.js frontend, built-in AI module, JWT auth, Stripe billing, and GDPR compliance.

## Quick Start

```bash
cp .env.example .env
make dev
# Backend API: http://localhost:8003/docs
# Frontend: make frontend-dev → http://localhost:3000
```
